> This repository contains the source code for the official website of the BIOML (Bioinformatics, Imaging & Omics using Machine Learning) Lab, hosted on GitHub Pages.

## Live Website

You can view the live lab website at:
### **[https://bioml-lab.github.io/](https://bioml-lab.github.io/)**

---

## Technology Stack

This is a fully static website built with modern, lightweight technologies. No complex frameworks or build steps are required.

-   **HTML5**
-   **CSS3**
-   **Bootstrap 5** for responsive design and core components.
-   **Vanilla JavaScript** for fade-in animations and the dark mode toggle.
-   **Font Awesome** for all icons.
-   **Formspree** for handling the contact form submissions.

---

## Deployment

This site is automatically deployed by **GitHub Pages**. Any changes pushed or saved to the `main` branch will be live within a few minutes.
